# customer-segmentation-dashboard
Customer Segmentation Dashboard: A Tableau dashboard that analyzes customer behavior, segments customers by value, and provides insights into sales trends and retention for data-driven business decisions.
Customer Segmentation Dashboard

This project is a Customer Segmentation Dashboard built using Tableau to help businesses analyze customer behavior and make data-driven decisions. The dashboard segments customers based on their purchase history into High, Medium, and Low Value categories and provides insights into revenue, retention, and seasonal trends.

Key features of the dashboard include:

Customer Segmentation: Categorizes customers based on total sales.

Interactive Visualizations: Explore sales trends, top customers, and geographic distribution.

Purchase Pattern Analysis: Understand buying behavior and identify high-value customers.

Actionable Insights: Supports strategic decisions for marketing, sales, and customer retention.

Dataset: Includes customer demographics, transaction details, and purchase history.

Tools Used: Tableau, Excel
